# ML-LAB — Machine Learning Algorithms From Scratch

This repository contains implementations of core Machine Learning algorithms built **from scratch in Python**, as part of the CS4089E – Machine Learning Laboratory (Winter 2025–26).

The objective of this lab is to deeply understand how machine learning algorithms work internally by implementing them without relying on high-level ML libraries like scikit-learn.

---

## 🎯 Course Information

- Course: CS4089E – Machine Learning Laboratory  
- Credits: 3  
- Institute: National Institute of Technology Calicut  
- Semester: Winter 2025–26  

---

## 📚 Objectives

- Implement machine learning algorithms from first principles.
- Understand mathematical foundations behind each algorithm.
- Analyze model performance and compare techniques.
- Apply ML algorithms to real-world datasets.
